# Blur Tool

A simple web-based manual blur utility. Upload an image, select an area to blur at your preferred intensity and download the result.

## Usage

Open `index.html` in your browser. Select an image file from your computer. Use the **Blur amount** slider to set the desired blur strength. Click and drag on the image to choose the region you want to blur. Release the mouse to apply the effect. Click **Download Result** to save the blurred image.
